# Zailon — NTE Addon for TwintailLauncher
Zailon adds Neverness to Everness (NTE) to your existing TwintailLauncher installation.
## Prerequisites
Install TwintailLauncher first: https://twintaillauncher.app
## Install
Windows: Right-click install.ps1 and run with PowerShell
Linux: chmod +x install.sh && ./install.sh
## Uninstall
Delete nte_global.json from your TTL manifests folder and remove its line from repository.json.
Windows: %APPDATA%\com.twintaillauncher.ttl\manifests\
Linux: ~/.local/share/com.twintaillauncher.ttl/manifests/
