# Zailon — NTE Addon for TwintailLauncher

Adds **Neverness to Everness (NTE)** to TwintailLauncher. Works like Spicetify for Spotify.

## Install

### Windows (PowerShell)
```powershell
iwr -useb https://raw.githubusercontent.com/Sankaiii/Zailon-Addon/stable/install.ps1 | iex
```

### Linux
```bash
curl -fsSL https://raw.githubusercontent.com/Sankaiii/Zailon-Addon/stable/install.sh | sh
```

## Prerequisites
Install TwintailLauncher first: https://twintaillauncher.app

## Uninstall
Delete nte_global.json from your TTL manifests folder.
